    <title>My First Page</title>

</html>

Python Syntax

Python syntax is known for being clean and easy to read. Unlike many other languages, it uses indentation to define code blocks.

Basic Syntax Rules

  • Indentation matters (uses whitespace to define scope)
  • No semicolons needed at the end of statements
  • Comments start with #
  • Variables are created when you first assign a value
  • Case sensitive
# Variables and data types
name = "Alice"  # String
age = 25        # Integer
height = 5.6    # Float
is_student = True # Boolean

# Conditional statement
if age >= 18:
    print(f"{name} is an adult")
else:
    print(f"{name} is a minor")

# For loop
for i in range(5):
    print(i)

# Function definition
def calculate_area(length, width):
    return length * width

Arduino Introduction

Arduino is an open-source electronics platform based on easy-to-use hardware and software.

What You Will Learn

  • Arduino board basics
  • Digital and analog I/O
  • Working with sensors
  • Serial communication
  • Building IoT projects
// the setup function runs once when you press reset or power the board
void setup() {
    // initialize digital pin LED_BUILTIN as an output
    pinMode(LED_BUILTIN, OUTPUT);
}

// the loop function runs over and over again forever
void loop() {
    digitalWrite(LED_BUILTIN, HIGH);  // turn the LED on
    delay(1000);                      // wait for a second
    digitalWrite(LED_BUILTIN, LOW);   // turn the LED off
    delay(1000);                      // wait for a second
}
